# spring-legacy-work

### MyBatis
- MyBatis는 자바 언어를 위한 오픈 소스 ORM(Object Relational Mapping) (객체 관계 맵핑) 프레임워크
- JDBC로 수행되는 데이터베이스 쿼리의 수고를 덜어주는 간편한 방법을 제공함

## Spring Legacy 에서의 MyBatis 사용 방법

Spring Framework에서 MyBatis를 사용하기 위해서는MyBatis와
